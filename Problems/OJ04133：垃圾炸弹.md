已知城市的大小，可以对整个城市进行遍历查找，但1025较大，需要考虑常数对运行时间的影响

若记录垃圾位置，对每个投弹位点，查找正方形范围内的垃圾，需要查找1025^2*d^2个位点

若将垃圾数量标记在可及的位点中，需要标记n*d^2个位点，查找1025^2个位点，这个方案更快

```python
d=int(input())
city=[[0]*1025 for _ in range(1025)]
n=int(input())
for _ in range(n):
    x,y,i=map(int,input().split())
    for r in range(max(0,x-d),min(1025,x+d+1)):
        for c in range(max(0,y-d),min(1025,y+d+1)):
            city[r][c]+=i
ans=(0,0)
for i in range(1025):
    for j in range(1025):
        if city[i][j]>ans[0]:
            ans=(city[i][j],1)
        elif city[i][j]==ans[0]:
            ans=(ans[0],ans[1]+1)
print(ans[1],ans[0])
```
