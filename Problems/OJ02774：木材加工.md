二分查找。运用左闭右开，因此最终的答案为hi-1。check函数x=0的情况要提前return True以免把0作为除数。

```python
N,K=map(int,input().split())
woods=[]
for _ in range(N):
    woods.append(int(input()))
def check(x):
    if x==0:
        return True
    count=0
    for wood in woods:
        count+=wood//x
    return count>=K
lo,hi=0,sum(woods)//K+1
while lo<hi:
    mid=(lo+hi)//2
    if check(mid):
        lo=mid+1
    else:
        hi=mid
print(hi-1)
```
