运用队列，每次向左旋转k-1次，把最左端的数字pop并加到答案中，直到队列中仅剩一个数字。

```python
from collections import deque
n,k=map(int,input().split())
queue=deque([x+1 for x in range(n)])
ans=[]
while len(queue)>1:
    queue.rotate(1-k)
    ans.append(queue.popleft())
print(*ans)
```

运用循环链表，每次a=a.next操作k-1次，把a所在的节点值加到答案中，把该节点删除，直到a.next=a。

```python
class Node:
    def __init__(self,val=0,next=None,last=None):
        self.val=val
        self.next=next
        self.last=last
def add(re_node,node):
    nxt=re_node.next
    re_node.next=node
    node.next=nxt
    node.last=re_node
    nxt.last=node
def delete(node):
    node.last.next=node.next
    node.next.last=node.last
n,k=map(int,input().split())
a=b=Node(1)
b.next=b
b.last=b
for i in range(1,n):
    add(b,Node(i+1))
    b=b.next
ans=[]
while a.next != a:
    for _ in range(k-1):
        a=a.next
    ans.append(a.val)
    c=a
    delete(c)
    a=a.next
print(*ans)
```
