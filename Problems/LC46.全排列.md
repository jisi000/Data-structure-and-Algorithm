递归。把数组的第一个数先拿出来，递归得到后面的数的所有可能排列，将第一个数依次插入到这些排列的所有位置。递归终止条件是数组中只有一个数。

```python
class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        if len(nums)==1:
            return [nums]
        n_nums=self.permute(nums[1:])
        ans=[]
        for l in n_nums:
            for i in range(len(l)+1):
                ans.append(l[:i]+[nums[0]]+l[i:])
        return ans
```
