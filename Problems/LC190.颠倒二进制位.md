完成整数转二进制数、二进制数颠倒、二进制数转整数三个操作即可

利用内置函数的简洁版

```python
class Solution:
    def reverseBits(self, n: int) -> int:
        return int(f'{n:032b}'[::-1],2)
```

运用位运算的方法

```python
class Solution:
    def reverseBits(self, n: int) -> int:
        ans=0
        for _ in range(32):
            bit=n&1
            n//=2
            ans=ans*2+bit
        return ans
```
