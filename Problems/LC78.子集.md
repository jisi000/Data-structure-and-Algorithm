对于每个元素，一半子集有该元素，一半子集没有。递归获得不包含这个元素的子集，全部的子集就是这些子集加上这个元素和不加这个元素的总和。

```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        if len(nums)==1:
            return [[],[nums[0]]]
        re=nums[0]
        sub_ans=self.subsets(nums[1:])
        ans=[]
        for l in sub_ans:
            ans.append(l)
            ans.append([re]+l)
        return ans
```
