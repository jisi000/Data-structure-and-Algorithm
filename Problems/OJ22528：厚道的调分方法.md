经典的二分查找题。

```python
scores=list(map(float,(input().split())))
n=len(scores)
def check(x):
    s=0
    for score in scores:
        if x*score/(10**9)+1.1**(x*score/(10**9))>=85:
            s+=1
    return s/n>=0.6
lo,hi=1,10**9+1
while lo<hi:
    mid=(lo+hi)//2
    if check(mid):
        hi=mid
    else:
        lo=mid+1
print(hi)
```
