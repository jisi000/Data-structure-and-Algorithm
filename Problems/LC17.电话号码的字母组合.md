用辅助列表re记录使用过的字母，当数字都使用过时，把re转为字符串储存在答案中。

```python
class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        d={'2':'abc','3':'def','4':'ghi','5':'jkl','6':'mno','7':'pqrs','8':'tuv','9':'wxyz'}
        ans,re=[],[]
        def backtrack(s):
            if len(s)==0:
                ans.append(''.join(re))
                return
            for ch in d[s[0]]:
                re.append(ch)
                backtrack(s[1:])
                re.pop()
        backtrack(digits)
        return ans
```
