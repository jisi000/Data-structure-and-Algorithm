利用长度固定为N的列表代表文档，‘1’代表这个词存在于此文档，‘-1’代表不存在

列表及其对应编号用字典存储，定义compare函数比较查询和文档是否相符

```python
N=int(input())
d={}
def compare(s1,s2):
    for i in range(N):
        if s2[i]!= '0' and s1[i]!=s2[i]:
            return False
    return True
for i in range(N):
    data=list(map(int,input().split()))
    s_data=set(data[1:])
    for doct in s_data:
        if doct not in d:
            d[doct]=['-1']*N
        d[doct][i]='1'
q=sorted(d.keys())
M=int(input())
for _ in range(M):
    inquire=list(input().split())
    ans=[]
    for j in q:
        if compare(d[j],inquire):
            ans.append(j)
    if ans:
        print(*ans)
    else:
        print('NOT FOUND')
```
