双指针。先把数组排序，两个指针放在左右两端。如果找到了T就直接输出，如果当前加和小于T，左指针+1，如果当前加和大于T，右指针-1。所有遍历到的加和值及其与T的差值的绝对值用元组（gap,num）存储，这样排序时优先考虑gap最小的，再考虑num的值较小的。

```python
def main():
    T=int(input())
    S=list(map(int,input().split()))
    S.sort()
    n=len(S)
    l,r=0,n-1
    ans=[]
    while l<r:
        if S[l]+S[r]==T:
            print(T)
            return
        elif S[l]+S[r]<T:
            ans.append((T-S[l]-S[r],S[l]+S[r]))
            l+=1
        else:
            ans.append((S[l]+S[r]-T,S[l]+S[r]))
            r-=1
    ans.sort()
    print(ans[0][1])
    return
if __name__=='__main__':
    main()
```
