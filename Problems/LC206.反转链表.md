h是当前节点，用re存储前面已经反转好的链表（re的初始值为None）。进行反转操作时，先用nxt暂存h的下一个节点，将当前节点h的下一个节点连接到re，然后把h赋值为nxt。当h=None时，已经反转好的链表头节点便是re

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        h=head
        re=None
        while h:
            nxt=h.next
            h.next=re
            re=h
            h=nxt
        return re
```
