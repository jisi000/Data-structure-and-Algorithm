先用字典存储每个队伍的提交数和AC数，然后用元组（-AC数，提交数，名称）排序

```python
m=int(input())
d1,d2={},{}
for _ in range(m):
    name,problem,yon=input().split(',')
    d2[name]=d2.get(name,0)+1
    if name not in d1:
        d1[name]=[0]*26
    if yon=='yes':
        d1[name][ord(problem)-65]=1
s=[]
for key,value in d1.items():
    s.append((-sum(value),d2[key],key))
s.sort()
for i in range(12):
    if i>=len(s):
        break
    print(i+1,s[i][2],-s[i][0],s[i][1])
```
