从两个头节点向前走，走到头就跳到另一个头节点，在两个节点走的步数相同的情况下，可以恰好在相交节点相遇。如果不存在相交节点，两节点会在None相遇。

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        a,b=headA,headB
        while a != b:
            a=a.next if a else headB
            b=b.next if b else headA
        return b
```
