每次获得n末位的数字，并把n右移一位，last_i用于记录上一次‘1’出现的位置，i记录当前的位是从右数第几位。如果当前n末位的数字是‘1’，就将i-last_i更新到答案中。

```python
class Solution:
    def binaryGap(self, n: int) -> int:
        ans=0
        last_i=-1
        i=0
        while n!=0:
            bit=n%2
            n>>=1
            if bit==1:
                if last_i!=-1:
                    ans=max(ans,i-last_i)
                last_i=i
            i+=1
        return ans
```
