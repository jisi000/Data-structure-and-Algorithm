二分查找，操作数 (用袋子球数-1)//开销 的累计值来计算

```python
class Solution:
    def minimumSize(self, nums: List[int], maxOperations: int) -> int:
        def check(x):
            count=0
            for i in nums:
                count+=(i-1)//x
            return count<=maxOperations
        lo=1
        hi=max(nums)+1
        while lo<hi:
            mid=(lo+hi)//2
            if check(mid):
                hi=mid
            else:
                lo=mid+1
        return hi
```
