先根据nums1给数组（nums1[i],nums2[i],i）排序。只需要将s(sum)初始设置为零，遍历过i之后将nums2[i]加到s中，这样遍历到i时的s值即为answer[i]。需要维护一个小顶堆，当s中元素数量超过k时，将最小的nums2[j]踢出去。

```python
class Solution:
    def findMaxSum(self, nums1: List[int], nums2: List[int], k: int) -> List[int]:
        import heapq
        num=sorted((x,y,i) for i,(x,y) in enumerate(zip(nums1,nums2)))
        n=len(num)
        ans=[0]*n
        s=0
        i=0
        h=[]
        while i<n:
            start=i
            x=num[i][0]
            while i<n and num[i][0]==x:
                ans[num[i][2]]=s
                i+=1
            for j in range(start,i):
                s+=num[j][1]
                heapq.heappush(h,num[j][1])
                if len(h)>k:
                    s-=heapq.heappop(h)
        return ans
```
