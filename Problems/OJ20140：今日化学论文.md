利用栈。当遇到 ‘]' 时，对前一个 ’[' 后面的字符进行整理，区分字母和数字部分，把字母部分*数字部分压入栈。

```python
s=input()
stack=[]
for ch in s:
    if ch !="]":
        stack.append(ch)
    else:
        re=''
        while stack:
            p=stack.pop()
            if p=="[":
                break
            re=p+re
        lo=0
        for i in range(len(re)):
            if re[i].isalpha():
                lo=i
                break
        if lo==0:
            stack.append(re)
        else:
            stack.append(re[lo:]*int(re[:lo]))
print(''.join(stack))
```
