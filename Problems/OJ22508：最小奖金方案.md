a战胜b，可转换为一条由b指向a的边，代表a的奖金要大于b。统计每个节点的入度和指向的其他节点。初始增值为0，代表只获得100元，统计入度为零的节点并将他们的增值加到总和中，每一轮增值+1。每一轮遍历都使用入度列表的复制体，避免本轮的入度修改导致不在本轮的节点入度归零，提前使用。

```python
from collections import defaultdict
n,m=map(int,input().split())
ind=[0]*n
d=defaultdict(list)
for _ in range(m):
    a,b=map(int,input().split())
    ind[a]+=1
    d[b].append(a)
total=0
re=0
visited=[False]*n
count=0
while count<n:
    re_ind=ind.copy()
    for i in range(n):
        if not visited[i] and re_ind[i]==0:
            count+=1
            total+=re
            visited[i]=True
            for j in d[i]:
                ind[j]-=1
    re+=1
print(n*100+total)
```
