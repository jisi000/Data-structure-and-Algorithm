因为数组非递减，相邻的数字都能连通，只有当相邻两个数的差值大于maxDIFF时，路径才会断掉。因此从0开始给节点赋予层数，每到断点层数加一，层数相同的的节点时相通的。

```python
class Solution:
    def pathExistenceQueries(self, n: int, nums: List[int], maxDiff: int, queries: List[List[int]]) -> List[bool]:
        rank=0
        layers=[]
        for i in range(n-1):
            layers.append(rank)
            if nums[i+1]-nums[i]>maxDiff:
                rank+=1
        layers.append(rank)
        ans=[]
        for i,j in queries:
            if layers[i]!=layers[j]:
                ans.append(False)
            else:
                ans.append(True)
        return ans
```
